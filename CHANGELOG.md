## [0.1.1] - 2025-11-29

### Added
- Initial release of the **Universal Week Calendar** package.
- Core widget implementation (`UniversalWeekCalendar`).
- Customizable properties for margin, padding, and theme colors.
- Callbacks for date selection (`onDateSelected`) and week navigation (`onNextWeek`, `onPreviousWeek`).
- Support for displaying a custom header text (e.g., month/year).
